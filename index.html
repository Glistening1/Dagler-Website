Exploring Computer Science

Shows high school students the vast world of computer science. Topics in this course include: human computer interaction, website design, developing problem solving techniques, different number systems, the command line interface, programming in the C language, and modular robotics. In the course several students will be given the opportunity to compete at a statewide robotics competition.

AP Computer Science A

Is equivalent to a first-semester, college-level course in computer science. This course emphasizes both object-oriented and imperative problem solving and design using the Java language. Some topics in this course include: design strategies and methodologies, data structures, developing algorithms, and approaches to processing data. This course is on a 5.0 GPA bump and there is an AP test the students can take at the end of the course for possible college credit.

AP Computer Science Principles

Gives students fundamental concepts of computer science and challenges them to explore how computing and technology is impacting us today. Students will use technology to address real-world problems while learning several topics including abstraction, beginning algorithms, large data sets, and cybersecurity concerns. This course is on a 5.0 GPA bump and there is an AP test the students can take at the end of the course for possible college credit.



Web Design and Development 1

This course teaches students to develop Web sites using Hypertext Markup Language version 5 (HTML5) and Cascading Style Sheets (CSS). They'll also insert images, create hyperlinks, and add tables, forms, videos and audio to Web pages, as well as using HTML5 Application Programming Interfaces (APIs) to extend the functionality of Web pages. Other topics include validating HTML code, recognizing the importance of search engine optimization (SEO), and implementing fundamental design concepts. Additional topics include an intro to JavaScript, Image Maps, BootStrap, SVG, and PGP.

Web Design and Development 2

This course teaches the essentials of HTML5, Cascading Style Sheets (CSS) and JavaScript (JS). Students will create Web pages using the HTML5 structure elements, embed video and audio, and develop cross-browser user-input forms. They will use CSS3 to position and format content, and to create effects such as transformations, transitions and animation. They will also learn basic JavaScript coding to extend the functionality of Web pages with modern features such as geolocation, drag-and-drop, canvas and offline Web applications. In addition, they will learn techniques for code validation and testing, form creation, inline form field validation, and mobile design for browsers and apps, including Responsive Web Design. Additional topics include FlexBox, CSS grid layout, intro to Google Analytics, SASS, and Node.js.

Industry Connections

This computer science pathway is more than just the classes you take, it also includes activities to connect with the industry. Some of the activities include: guest speakers, field trips, working with local companies on a project, and computions.
